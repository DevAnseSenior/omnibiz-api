# OmniBiz API

## 📋 Project Description
A modern Enterprise Resource Planning (ERP) system built with **Django REST Framework** for efficient business management.

## Features

- Modular Design: Manage finance, inventory, sales, and HR in one platform.

- RESTful API: Scalable backend for web/mobile clients.

- Database: MySQL (or MySQL) for reliable data storage.

## 🛠️ Technologies

- Backend: Django REST Framework (DRF)

- Database: MySQL

## 🔧 Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/DevAnseSenior/omnibiz-api.git
   cd omnibiz-api
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt 
   ```

4. Run migrations:
   ```bash
   python manage.py migrate  
   ```
   
5. Start the server:
   ```bash
   python manage.py runserver
   ```

## 📄 API documentation:

   - Interactive API docs (Swagger UI): `http://127.0.0.1:8000/docs`
